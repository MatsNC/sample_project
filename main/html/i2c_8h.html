esp_err_t i2c_master_read_slave ( uint8_t * data,
size_t len )

Lee datos del esclavo a través de I2C.

Esta función lee un array de bytes desde el esclavo utilizando la interfaz I2C.

Parameters
[out]dataPuntero al array donde se almacenarán los datos leídos.
[in]lenLongitud del array de datos a leer.
Returns
  • ESP_OK si los datos fueron leídos exitosamente.
  • Otro valor de error de esp_err_t si ocurrió un error durante la recepción.
